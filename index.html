Open Source Community Health System

Our Goal

OpenCHS aims to be a comprehensive community health platform designed to empower health workers with supportive supervision of the clinicians. It is being developed in collaboration with practitioners working in the field with large experience in community medicine.

OpenCHS team started work on it in the second half of 2016. The first feature being developed is a decision support platform customisable for the health workers in the field.

Key Technologies

Mobile

Even though web view kind of interface on mobile have come a long way they still do not match native app experience, especially on low end devices in trying conditions. We chose React Native because it is easy to learn, speeds up development cycle and will allow us to peform play store less upgrade.

We are using Realm as our offline database.

Infrastructure

We are managing our infrastructure using docker (honestly, most of work right now has been on mobile side)

Source Code

OpenCHS Client, OpenCHS Server can be found on Github

Team

The founding members & developers on Bahmni (the open source hospital system) have started this effort. We are very keen to involve the community in development and implementation of OpenCHS. Currently we are in the process of setting up necessary infrastructure so that we can involve the community.

Some of the high level features planned

  • Decision Support Platform
  • Individual and household registration
  • Followup visit tracking
  • Supportive supervision
  • Integration with DHIS 2 and OpenMRS

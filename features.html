OpenCHS Features

This is a current list of features that are available in OpenCHS. We update the system frequently. A detailed roadmap of OpenCHS is available here.

Alternatively, download the Android app from the playstore and try it out yourself.

General features

  • Individual search and registration
  • Enrolment into programs
  • Dashboards for health worker
  • Individual dashboard
  • Offline android app, sync facility to server

Health modules

Mother program

  • Plan mandatory ANC visits with structured data collection on each visit
  • Capture ANC, delivery, PNC details on device
  • Automatic high risk identification

Child program

  • View and capture growth information on device
  • View and capture vaccination details on device
  • View growth chart (WHO) on device

Outpatient/screening

  • Support for medication for 16 minor ailments
